# zh-xmty-guide

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容相互独立，不针对任何特定域名或网站。仓库旨在以静态方式提供可浏览的 HTML 资料集。

## 项目简介

zh-xmty-guide 是一个以 HTML 页面为主要载体的归档仓库。仓库中的每个 HTML 文件均为独立单元，可直接在浏览器中打开查看。本仓库不提供动态服务，仅作为静态文件存储与分发使用。

## 目录说明

```
zh-xmty-guide/
├── index.html          # 仓库首页，展示页面索引
├── pages/              # 存放独立 HTML 页面的目录
│   ├── page-example.html
│   └── ...
├── assets/             # 公共资源文件（如 CSS、JS、图片等）
│   ├── style.css
│   └── ...
└── README.md           # 本文件
```

- `index.html`：仓库的入口页面，通常包含页面导航或链接列表。
- `pages/`：核心内容目录，所有独立 HTML 页面存放于此。每个页面可包含自身所需的样式与脚本。
- `assets/`：存放页面间共享的静态资源，便于统一维护。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接在浏览器中运行。
- 页面之间无强依赖关系，可单独访问或分发。
- 页面内容可能包含文本、图像、样式、脚本等元素，但均以本地或相对路径引用。
- 本仓库不保证所有页面内容持续更新或长期有效。
